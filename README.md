# Dashboard
[![Build Status](https://travis-ci.com/LukeANewton/personal-dashboard.svg?branch=master)](https://travis-ci.com/LukeANewton/personal-dashboard)

A web application to collect internet content of interest. It can be found [here](https://dashboard-for-stuff.herokuapp.com/).

## Built With
- [React](https://reactjs.org/) - JavaScript library for front end
- [Express](https://expressjs.com/) - JavaScript (Node) framework for back end
- [Travis CI](https://docs.travis-ci.com/) - Continuous Integration service
- [Heroku](https://devcenter.heroku.com/) - Cloud service to host application
